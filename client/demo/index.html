User Roles & Permissions

Role-based access control with 3 user types

Administrator

Full system access

Capabilities

  • All product management operations
  • All inventory operations
  • User management (create, lock, delete)
  • System settings configuration
  • View all audit logs
  • Access all warehouses

Warehouse Manager

Operational access

Capabilities

  • View all products
  • Manage inventory for assigned warehouse
  • Create and manage stock transfers
  • Approve/reject transfers
  • Receive goods (GRN)
  • View and acknowledge alerts

Inventory Clerk

Limited operational access

Capabilities

  • View products and stock levels
  • Perform stock adjustments
  • Participate in stock counts
  • View movement history
  • Acknowledge alerts
